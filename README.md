# YYA.OnionArchitecture
:wave:
## Introduction
This solution contains a .NET Core app built on the onion architecture with some extra features
## Features
- .NET Core 7
- EF Core 7
- AutoMapper
- FluentValidation
- CQRS and Mediator pattern via MediatR
- Middlewares
- Unit Testing via Nunit , FluentAssertions and Moq
- Swagger
- Logging
